# Step 04: Present Results

## RULES
- YOU MUST ALWAYS SPEAK OUTPUT in your Agent communication style with the config `{communication_language}`
- Keep the presentation concise, actionable, and professional.

## INSTRUCTIONS

1.  **Draft Report:** Compile the triaged findings into a final report.
2.  **Summary:** Start with a brief summary of the review:
    - Number of findings by severity (Critical/Major/Minor).
    - Overall state of the code.
3.  **Findings Table:** Present the triaged findings in a table with:
    - Severity
    - Category
    - Description
    - Evidence/File Path
4.  **Action Plan:** Conclude with the recommended next steps.

**After presenting, ask the user if there are any questions before finalizing the review.**
